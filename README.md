# Registro_usuario
Registro de usuario, realizado en HTML, CSS, BOOTSTRAP, PHP Y MYSQL, Password protegida por md5.


Registro de usuario simple.  para poder usar el código solo tienes que crear  una base de datos

ir al archivo config.php  y  cambiar los datos de acuerdo a tu base de datos

$ servidor = "nombre_servidor" ;
$ usuario = "nombre de usuario, generalmente es root" ;
$ password = "si tiene contraseña, si no va vacío" ;
$ base de datos = "nombre_bd" ;

$ conn = mysqli_connect ( $ servidor , $ usuario , $ contraseña , $ base de datos );


una vez cambiado  queda totalmente funcional  y se puede visualizar en tu basde de datos
